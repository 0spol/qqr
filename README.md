## Features

* Terminal-based QR code generation
* Multiple encoding modes: Byte, Numeric, Alphanumeric, Kanji, Hanzi
* Adjustable error correction levels: L, M, Q, H
* Export options: PNG, SVG, PDF, ASCII
* Lightweight and easy to use
